# ЭРМ маршруты

Web-приложение на FastAPI для автоматической обработки Excel-файлов маршрутов и формирования справочника параметров маршрутов без ручного ввода протяженности и времени рейса.

## Возможности

- загрузка одного или нескольких `.xlsx`;
- рекурсивная обработка папки на сервере;
- поиск вкладки `параметры` без учета регистра;
- извлечение номера, названия маршрута, остановок, координат, улиц, протяженности, времени рейса и средней скорости;
- проверка ошибок и предупреждений;
- поиск дублей маршрутов и расхождений;
- хранение истории ручных исправлений и подтверждений актуальной версии;
- экспорт отчета Excel с листами `Справочник маршрутов`, `Ошибки и предупреждения`, `Дубли и расхождения`, `Остановки`, `Журнал обработки`, `История изменений`.

## Быстрый запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Откройте `http://127.0.0.1:8000`.

Если команда `uvicorn` не найдена, запустите сервер так:

```bash
python -m uvicorn app.main:app --reload
```

Если виртуальное окружение не создалось, не выполняйте `pip install` в системный Python. Сначала установите доступную версию Python и заново создайте `.venv`.

## Docker

```bash
docker compose up --build
```

Приложение будет доступно на `http://127.0.0.1:8000`.

## Структура

- `app/main.py` - FastAPI endpoints и серверный web-интерфейс.
- `app/parser.py` - гибкий разбор Excel-файлов через openpyxl.
- `app/models.py` - SQLite-модели SQLAlchemy.
- `app/services.py` - сохранение маршрутов, история, дубли и подтверждения.
- `app/exporter.py` - Excel-отчет через openpyxl.
- `app/templates/index.html` - основной интерфейс.
- `app/static/` - CSS и JavaScript.
- `tests/test_parser.py` - тесты ключевой логики разбора.

## Переменные окружения

- `DATABASE_URL` - строка подключения SQLAlchemy, по умолчанию `sqlite:///data/erm.db`.
- `UPLOAD_DIR` - папка для загруженных файлов, по умолчанию `uploads`.

SQLite используется на первом этапе. Для PostgreSQL достаточно поменять `DATABASE_URL` и установить драйвер, например `psycopg`.
